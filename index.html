Voice Lines

This section is for configuring CASSIE, Announcements and other SCP:RP Voice Lines.

The way we are going to acheive this, is a similar process to how VOX from HL1 works. We are going to store each .wav file we need, and set up ordered Arrays to play each clip one after another.

These .wav files are referred to as Clips throughout this application.

The UI below will allow you to add new Clips, so we know what to add to the server.
